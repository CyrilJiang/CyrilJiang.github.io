历史渊源

  • C++基于 C

  • Java 基于 C 和 C++ ,且取消了指针和内存管理

  • Java 适合编写大型程序

    • 高可用
    • 高并发
    • 高性能

  • 大型工程开发的工具

    • 构建工具：Ant、Maven、Jekins
    • 应用服务器：Tomcat、Jetty、Jboss、Websphere、weblogic
    • web 开发：Struts、spring、hibernate、mybatis
    • 开发工具：eclipse、NetBean、intellij idea、Jbuilder

  • 简单：比 C++简单，比 C 简单？

  • 动态性：反射

  • 安全性

继承

  1. 是对一些类的进一步抽象

  2. 快捷键 <kbd>Ctril </kbd>+<kbd>H </kbd>打开继承视图

    1. 工具栏 Navigate->Type Hierarchy

  3. java 中只能单继承

  4. super 关键字（父类的对象）

    1. super()调用父类的构造方法，必须在构造方法的第 1 个
    2. super()必须只能出现在子类的方法或者构造方法中
    3. super()和 this()不能同时调用构造方法

  5. this 关键字（本类的对象）

    1. 未继承也可以用
    2. this()本类的构造方法
重写

  1. 子类继承父类，可以重写父类方法

    1. 万法名必须相同
    2. 参数列表列表必须相同
    3. 修饰符:范围可以扩大但不能缩小: public>Protected>Default>private
    4. 抛出的异常:范围，可以被缩小，但不能扩大; ClassNotFoundException –> Exception(大)

  2. 不可重写的情况

    1. static 方法，属于类，它不属于实例
    2. final 常量;
    3. 3.private 方法;

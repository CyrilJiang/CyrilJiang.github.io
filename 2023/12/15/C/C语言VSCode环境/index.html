C 语言 VSCode 环境

实现 VSCode 调试 C 代码，作为学习环境

windows 系统 C 编译器

各种编译器的差别参考C/C++标准库，常见的有：

  1. MSYS2：包含 clang，MinGW，ucrt，体积大
  2. MinGW-w64：Minimalist GNU for Windows，体积较小

windows 系统安装 MinGW

  1. 下载解压
    下载MinGW-w64 x86_64-posix-seh 版本，解压到”D:\mingw64“。
  2. 添加环境变量
    右击我的电脑-添加环境变量“D:\mingw64\bin”
  3. 检查是否生效
    cmd.exe 输入 gcc -v

windows 系统配置 VSCode

安装 C/C++ Extension Pack 扩展，按照 Get Started with C++ and MinGW-w64 in Visual Studio Code配置即可。

可能的错误

  1. 红色波浪线
    说明 intellisense 检查语法时未识别符号，应检查 c_cpp_properties.json 里面的”includePath”标签，该标签是 intellisense 检查时使用的路径。

  2. 编译时出现未找到 xxx .h
    说明编译器未找到头文件，应检查 tasks.json 里面的”args”标签，该标签描述 gcc 的编译参数。应添加额外引用目录：

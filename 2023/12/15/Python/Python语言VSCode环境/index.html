Python 语言 VSCode 环境

实现 VSCode 调试 Python 代码，作为学习环境

windows 系统 Python 解释器

Python 官网提供两种安装方式：

  1. embeddable package.zip：嵌入式版本，不带 pip，setuptools，wheel，压缩包形式，体积较小
  2. executable installer.exe：可执行程序，完整版，可执行程序，体积大

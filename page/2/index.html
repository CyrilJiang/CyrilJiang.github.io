内网穿透树莓派

内网穿透树莓派

[toc]

目的

商业服务器需要花钱购买。那么有没有免费的服务器呢？答案是有的。就是本期的主角：树莓派 3B(当然 4B 也是一样的)。
我们使用树莓派作为服务器，并且使用内外穿透工具。使得服务端口的暴露到公网上，实现随时随地访问家庭服务器。

树莓派 3B

树莓派 3B 主要有以下优点：

  1. 功耗低
    树莓派 3B 的默频功耗在 5W 左右。以下是浙江省电网销售电价表的部分截取内容

    居民生活用电 电度电价 高峰电价 低谷电价
    年用电 2760 千瓦时及以下部分 0.5380 0.5680 0.2880

    居民生活用电分时电价时段划分：高峰时段 8：00-22：00，低谷时段 22：00-次日 8：00

    我们可以得到一年的电费为：

    $$
    电费 = 0.005kW24H365Day*0.5380元/kW·h = 23.5644元
    $$

  2. 价格便宜
    树莓派 3B 目前在某鱼二手价格 100 元左右

  3. 学习资源丰富
    入门教程，包教包会。树莓派实验室

  4. 支持各种镜像
    国内系统镜像，流程下载体验。清华大学开源软件镜像站

  5. 来电自启动
    避免家中断电失联的尴尬情况

前期工作

装系统：Windowns 下安装树莓派系统到 SD 卡
联网：无屏幕和键盘配置树莓派 WiFi 和 SSH

花生壳免费穿透内网

  1. 首先下载 Raspberry Pi 32 位版本的花生壳客户端
  2. 使用 MobaXterm 自带的 sftp 服务，上传客户端到树莓派中
  3. 按照教程安装：花生壳 5.0 for 树莓派

手动挡驾车总结

手动挡驾车总结

[toc]

起步熄火

平地一档起步（易）

  • 不踏脚刹时，离合器起得太快，阻力太大
  • 踩脚刹时，此时虽然离合器起得不快，但是受脚刹影响
  • 合器抬起时车身抖动，此时如果不放脚刹照样熄火

坡地一档起步（中等）

  • 受车身重力影响，此时起步转速不足以带动受车身自重阻力影响下的车辆
  • 此时，必须稍稍加油门同时缓慢抬起离合器，若不加油门则熄火

台阶一档起步（难）

  • 此时车辆必须克服车重做功，抬升一定高度
  • 此时，必须稍稍加油门同时缓慢抬起离合器，若不加油门则熄火

总结

熄火原因都是由于阻力太大引起的。

侧放停车

  1. 因为人坐在车内，视野受限。必须参照旁边物体，使得车身与参照物平行。
  2. 然后方向打死，即可倒车入库
  3. 在停车过程中，通过调整方向盘，使得车身与车库的墙壁（白线）平行。
  4. 继续倒车，直至车尾贴近车库（白线）尾端，停车。

总结

倒车时方向与方向盘的转向是相反的

  • 前进时，方向盘转的方向就是车头转向。
  • 后退时，方向盘转向是车尾转向。

换挡

首先离合器踩到底，然后换挡，还完缓慢松开离合器

刹车

挺稳时使用手刹，完全抱死车轮。
刹车时，若遇到车身震动，可踏下离合器，较少阻力，避免熄火。

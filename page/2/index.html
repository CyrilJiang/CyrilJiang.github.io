云服务器选购

云服务器选购

[toc]

目前接触到的云服务器:网页空间、VPS（虚拟专用主机）、弹性云服务器（比如:腾讯轻量应用服务器，阿里云 ECS 服务器）、挂机宝、个人物理机、裸金属服务器

类型 配置 独立 IP 价格
网页空间 1 H 1GB，网页空间 1GB，数据库 200MB 无 12 元/年
VPS 1 H 1GB，20G 系统盘，1Gb/s 有 9.9$/年
弹性云 1 H 1GB，20G 系统盘，3Mb/s 有 99 元/年
挂机宝 2 H 1GB，10G 系统盘，100Mb/s 无 5 元/月
个人物理机 4 H 8GB，250G 系统盘，100Mb/s 无 125 元/月
裸金属服务器 8 H 32GB，2T 系统盘，100Mb/s 有 1000 元/月

网页空间

实现原理未知。一般提供一定的 CPU 和内存，网页空间容量 1GB，以及数据库 200MB。用作个人动/静态网页的托管。

  1. 需求侧
    最低价格:12 元/年
    技术要求:网页开发技能
    开放功能:网页托管环境 apache 等，ftp 上传，数据库 mysql，网站备份功能
    推荐:

    • ftp 客户端:FileZilla
    • 供应商:悍铭

  2. 供应侧
    维护成本:一台物理机甚至虚拟机可以分成很多个网页空间，平分到单用户的成本极低
    技术难度:未知

VPS

通过传统的虚拟化技术（kvm，等）创建的服务器。服务器硬件资源固定且不可更改。国内的 VPS 一般带宽低，只能做常规应用。国外的 VPS 共享带宽可达 1Gb/s，所以有很多人买了做网关转发，可架设 SSR。做成机场盈利。

  1. 需求侧
    最低价格:9.9$/年
    技术要求:服务器维护知识
    开放功能:默认 ssh 端口，root 权限，后续自己开放其他端口
    推荐:

    • ssh 客户端:MobaXterm
    • 供应商:RackNerd

  2. 供应侧
    维护成本:一台物理机可以根据自有资源拆分成多个 VPS，平分到单用户的成本较高
    技术难度:虚拟化技术

弹性云服务器

基于 IasS 技术，可扩容，带宽低。阿里、腾讯等大厂提供应用镜像源和常用系统镜像。

  1. 需求侧
    最低价格:99 元/年
    技术要求:服务器维护知识
    开放功能:默认 ssh 端口，root 权限，后续自己开放其他端口
    推荐:

    • ssh 客户端:MobaXterm
    • 供应商:腾讯云

  2. 供应侧
    维护成本:机房维护
    技术难度:使用云化技术实现计算资源动态分配

挂机宝

提供 win 环境的低配置虚拟机，一般无公网 IP。只能够挂 QQ 机器人，千牛等客服软件挂机服务，2D 网页游戏挂机。

  1. 需求侧
    最低价格:5 元/月
    技术要求:远程连接
    开放功能:默认 3389 端口
    推荐:

    • 远程控制软件:mstsc，Todesk
    • 供应商:闲鱼

  2. 供应侧
    维护成本:个人闲置计算机，无需维护
    技术难度:没难度

个人物理机

个人电脑远程租给客户，一般无公网 IP，带独立显卡。游戏开私服、3D 游戏、挂机。

  1. 需求侧
    最低价格:125 元/月
    技术要求:远程连接
    开放功能:默认 3389 端口
    推荐:

    • 远程控制软件:mstsc，Todesk
    • 供应商:闲鱼

  2. 供应侧
    维护成本:个人闲置计算机，无需维护
    技术难度:没难度

裸金属服务器

托管于机房的实体物理机，性能强悍，独占物理机资源。

  1. 需求侧
    最低价格:1000 元/月
    技术要求:服务器维护知识
    开放功能:默认 ssh 端口，root 权限，后续自己开放其他端口
    推荐:

    • ssh 客户端:MobaXterm
    • 供应商:RackNerd

  2. 供应侧
    维护成本:机房维护
    技术难度:只需提供远程控制技术

配置选择

  1. 核心数
    1H
  2. 内存
    Linux 1G，win 2G 起步
  3. 带宽
    建站:参考云主机如何选带宽和购买服务器时如何选择服务器带宽，在没 CDN 的情况下，2M 足够在 5 秒以内打开纯网页，大并发访问的情况需要 CDN，视频网站建站成本更高。同流量下，CDN 的成本远低于升级虚拟机配置。多用户时，购买 CDN 服务。可以使用第三方云盘提供客户下载选项，减少成本。家庭 100M 带宽会有 30M 的下行带宽，适合建站
    SSR:50Mbps 看视频才不卡，流量大于等于 300GB/月
    内网穿透:8Mbps 能流畅办公
  4. 存储
    建站:参考网站空间服务器大小选择，不做大型网站，不做视频网站。1G 足够用了。
    SSR:无需求
    内网穿透:无需求

总结

从技术的角度来讲，以后会逐步淘汰虚拟化技术，转向云技术+云存储技术，合理充分利用计算机性能。

  1. 搭建博客:网页空间，价格便宜，机器稳定，内容不易丢失
  2. 挂机 3D 游戏:租用个人物理机，性能足，价格相对便宜
  3. 饥荒服务器:云服务器(ECS)、也可以选择购买第三方服务
  4. 玩游戏:云游戏平台，目前斗鱼云游戏是免费玩
  5. 电商/QQ 机器人:挂机宝
  6. 企业应用:裸金属服务器、自建机房、云服务器(ECS)

内网穿透树莓派

内网穿透树莓派

[toc]

目的

商业服务器需要花钱购买。那么有没有免费的服务器呢？答案是有的。就是本期的主角：树莓派 3B(当然 4B 也是一样的)。
我们使用树莓派作为服务器，并且使用内外穿透工具。使得服务端口的暴露到公网上，实现随时随地访问家庭服务器。

树莓派 3B

树莓派 3B 主要有以下优点：

  1. 功耗低
    树莓派 3B 的默频功耗在 5W 左右。以下是浙江省电网销售电价表的部分截取内容

    居民生活用电 电度电价 高峰电价 低谷电价
    年用电 2760 千瓦时及以下部分 0.5380 0.5680 0.2880

    居民生活用电分时电价时段划分：高峰时段 8：00-22：00，低谷时段 22：00-次日 8：00

    我们可以得到一年的电费为：

    $$
    电费 = 0.005kW24H365Day*0.5380元/kW·h = 23.5644元
    $$

  2. 价格便宜
    树莓派 3B 目前在某鱼二手价格 100 元左右

  3. 学习资源丰富
    入门教程，包教包会。树莓派实验室

  4. 支持各种镜像
    国内系统镜像，流程下载体验。清华大学开源软件镜像站

  5. 来电自启动
    避免家中断电失联的尴尬情况

前期工作

装系统：Windowns 下安装树莓派系统到 SD 卡
联网：无屏幕和键盘配置树莓派 WiFi 和 SSH

  1. 首先下载 Raspberry Pi 32 位版本的花生壳客户端
  2. 使用 MobaXterm 自带的 sftp 服务，上传客户端到树莓派中
  3. 按照教程安装：花生壳 5.0 for 树莓派

手动挡驾车总结

手动挡驾车总结

[toc]

起步熄火

平地一档起步（易）

  • 不踏脚刹时，离合器起得太快，阻力太大
  • 踩脚刹时，此时虽然离合器起得不快，但是受脚刹影响
  • 合器抬起时车身抖动，此时如果不放脚刹照样熄火

坡地一档起步（中等）

  • 受车身重力影响，此时起步转速不足以带动受车身自重阻力影响下的车辆
  • 此时，必须稍稍加油门同时缓慢抬起离合器，若不加油门则熄火

台阶一档起步（难）

  • 此时车辆必须克服车重做功，抬升一定高度
  • 此时，必须稍稍加油门同时缓慢抬起离合器，若不加油门则熄火

总结

熄火原因都是由于阻力太大引起的。

侧放停车

  1. 因为人坐在车内，视野受限。必须参照旁边物体，使得车身与参照物平行。
  2. 然后方向打死，即可倒车入库
  3. 在停车过程中，通过调整方向盘，使得车身与车库的墙壁（白线）平行。
  4. 继续倒车，直至车尾贴近车库（白线）尾端，停车。

总结

倒车时方向与方向盘的转向是相反的

  • 前进时，方向盘转的方向就是车头转向。
  • 后退时，方向盘转向是车尾转向。

换挡

首先离合器踩到底，然后换挡，还完缓慢松开离合器

刹车

挺稳时使用手刹，完全抱死车轮。
刹车时，若遇到车身震动，可踏下离合器，较少阻力，避免熄火。

windows 系统 C 编译器

各种编译器的差别参考C/C++标准库，常见的有：

  1. MSYS2：包含 clang，MinGW，ucrt，体积大
  2. MinGW-w64：Minimalist GNU for Windows，体积较小

windows 系统安装 MinGW

  1. 下载解压
    下载MinGW-w64 x86_64-posix-seh 版本，解压到”D:\mingw64“。
  2. 添加环境变量
    右击我的电脑-添加环境变量“D:\mingw64\bin”
  3. 检查是否生效
    cmd.exe 输入 gcc -v

windows 系统配置 VSCode

安装 C/C++ Extension Pack 扩展，按照 Get Started with C++ and MinGW-w64 in Visual Studio Code配置即可。

  1. 红色波浪线
    说明 intellisense 检查语法时未识别符号，应检查 c_cpp_properties.json 里面的”includePath”标签，该标签是 intellisense 检查时使用的路径。

  2. 编译时出现未找到 xxx .h
    说明编译器未找到头文件，应检查 tasks.json 里面的”args”标签，该标签描述 gcc 的编译参数。应添加额外引用目录：

XML基础知识.md

XML 基础知识

[TOC]

简介

  • XML（Extensible Markup Language,可扩展标记语言）
  • XML 被设计用来传输和存储数据
  • 不是一种编程语言
  • 新增元素不会引起程序崩溃
  • 任意深度子元素

语言类比

  • XML 和 markdown 和 latex 一样记录内容，但不是所见即所得

  • XML 和 json 类似，都是数据交互格式
    - xml 是重量级的，json 是轻量级的

    • xml 在传输过程中比较占带宽，json 占带宽少，易于压缩
    • xml 和 json(JavaScript Object Notation, JS 对象简谱) 都用在项目交互下，xml 多用于做配置文件，json 用于数据交互
    • json 可用 jackson，gson 等方法解析，xml 可用 dom，sax，demo4j 等方式解析

  1. 所有 XML 元素都须有关闭标签

  2. 若元素为空，可以折叠标签

  3. XML 标签对大小写敏感

  4. XML 必须正确地嵌套

  5. XML 文档必须有根元素

  6. XML 的属性值须加单/双引号

  7. 实体引用(C 语言中的转义字符)

    引用（转义）符号 显示符号 中文名
    &lt; < 小于
    &gt; > 大于
    &amp; & 和号
    &apos; ‘ 单引号
    &quot; “ 引号

  9. 在 XML 中，空格会被保留

  10. XML 以 LF 存储换行

元素的结构

  1. 元素

  2. 本文和属性

元素命名规则

  • 名称可以含字母、数字以及其他的字符
  • 名称不能以数字或者标点符号开始
  • 名称不能以字符 “xml”（或者 XML、Xml）开始
  • 名称不能包含空格

最佳命名习惯

  • 避免 “-“ 字符
  • 避免 “.” 字符
  • 避免 “:” 字符

属性和元素

  • 尽量使用子元素而非属性

    属性和子元素元素可以提供同样的信息

    • 属性无法包含多重的值（元素可以）
    • 属性无法描述树结构（元素可以）
    • 属性不易扩展（为未来的变化）
    • 属性难以阅读和维护

  • 元数据（有关数据的数据）应当存储为属性，而数据本身应当存储为元素

XML 语法验证

  • 拥有正确语法的 XML 被称为“形式良好”的 XML
  • 通过 DTD 验证的 XML 是“合法”的 XML
    • DTD 的作用是定义 XML 文档的结构。它使用一系列合法的元素来定义文档结构。
    • 只要把 DOCTYPE 声明添加到代码中即可

云服务器选购.md

云服务器选购

[toc]

目前接触到的云服务器:网页空间、VPS（虚拟专用主机）、弹性云服务器（比如:腾讯轻量应用服务器，阿里云 ECS 服务器）、挂机宝、个人物理机、裸金属服务器

类型 配置 独立 IP 价格
网页空间 1 H 1GB，网页空间 1GB，数据库 200MB 无 12 元/年
VPS 1 H 1GB，20G 系统盘，1Gb/s 有 9.9$/年
弹性云 1 H 1GB，20G 系统盘，3Mb/s 有 99 元/年
挂机宝 2 H 1GB，10G 系统盘，100Mb/s 无 5 元/月
个人物理机 4 H 8GB，250G 系统盘，100Mb/s 无 125 元/月
裸金属服务器 8 H 32GB，2T 系统盘，100Mb/s 有 1000 元/月

网页空间

实现原理未知。一般提供一定的 CPU 和内存，网页空间容量 1GB，以及数据库 200MB。用作个人动/静态网页的托管。

  1. 需求侧
    最低价格:12 元/年
    技术要求:网页开发技能
    开放功能:网页托管环境 apache 等，ftp 上传，数据库 mysql，网站备份功能
    推荐:

    • ftp 客户端:FileZilla
    • 供应商:悍铭

  2. 供应侧
    维护成本:一台物理机甚至虚拟机可以分成很多个网页空间，平分到单用户的成本极低
    技术难度:未知

VPS

通过传统的虚拟化技术（kvm，等）创建的服务器。服务器硬件资源固定且不可更改。国内的 VPS 一般带宽低，只能做常规应用。国外的 VPS 共享带宽可达 1Gb/s，所以有很多人买了做网关转发，可架设 SSR。做成机场盈利。

  1. 需求侧
    最低价格:9.9$/年
    技术要求:服务器维护知识
    开放功能:默认 ssh 端口，root 权限，后续自己开放其他端口
    推荐:

    • ssh 客户端:MobaXterm
    • 供应商:RackNerd

  2. 供应侧
    维护成本:一台物理机可以根据自有资源拆分成多个 VPS，平分到单用户的成本较高
    技术难度:虚拟化技术

弹性云服务器

基于 IasS 技术，可扩容，带宽低。阿里、腾讯等大厂提供应用镜像源和常用系统镜像。

  1. 需求侧
    最低价格:99 元/年
    技术要求:服务器维护知识
    开放功能:默认 ssh 端口，root 权限，后续自己开放其他端口
    推荐:

    • ssh 客户端:MobaXterm
    • 供应商:腾讯云

  2. 供应侧
    维护成本:机房维护
    技术难度:使用云化技术实现计算资源动态分配

挂机宝

提供 win 环境的低配置虚拟机，一般无公网 IP。只能够挂 QQ 机器人，千牛等客服软件挂机服务，2D 网页游戏挂机。

  1. 需求侧
    最低价格:5 元/月
    技术要求:远程连接
    开放功能:默认 3389 端口
    推荐:

    • 远程控制软件:mstsc，Todesk
    • 供应商:闲鱼

  2. 供应侧
    维护成本:个人闲置计算机，无需维护
    技术难度:没难度

个人物理机

个人电脑远程租给客户，一般无公网 IP，带独立显卡。游戏开私服、3D 游戏、挂机。

  1. 需求侧
    最低价格:125 元/月
    技术要求:远程连接
    开放功能:默认 3389 端口
    推荐:

    • 远程控制软件:mstsc，Todesk
    • 供应商:闲鱼

  2. 供应侧
    维护成本:个人闲置计算机，无需维护
    技术难度:没难度

裸金属服务器

托管于机房的实体物理机，性能强悍，独占物理机资源。

  1. 需求侧
    最低价格:1000 元/月
    技术要求:服务器维护知识
    开放功能:默认 ssh 端口，root 权限，后续自己开放其他端口
    推荐:

    • ssh 客户端:MobaXterm
    • 供应商:RackNerd

  2. 供应侧
    维护成本:机房维护
    技术难度:只需提供远程控制技术

配置选择

  1. 核心数
    1H
  2. 内存
    Linux 1G，win 2G 起步
  3. 带宽
    建站:参考云主机如何选带宽和购买服务器时如何选择服务器带宽，在没 CDN 的情况下，2M 足够在 5 秒以内打开纯网页，大并发访问的情况需要 CDN，视频网站建站成本更高。同流量下，CDN 的成本远低于升级虚拟机配置。多用户时，购买 CDN 服务。可以使用第三方云盘提供客户下载选项，减少成本。家庭 100M 带宽会有 30M 的下行带宽，适合建站
    SSR:50Mbps 看视频才不卡，流量大于等于 300GB/月
    内网穿透:8Mbps 能流畅办公
  4. 存储
    建站:参考网站空间服务器大小选择，不做大型网站，不做视频网站。1G 足够用了。
    SSR:无需求
    内网穿透:无需求

总结

从技术的角度来讲，以后会逐步淘汰虚拟化技术，转向云技术+云存储技术，合理充分利用计算机性能。

  1. 搭建博客:网页空间，价格便宜，机器稳定，内容不易丢失
  2. 挂机 3D 游戏:租用个人物理机，性能足，价格相对便宜
  3. 饥荒服务器:云服务器(ECS)、也可以选择购买第三方服务
  4. 玩游戏:云游戏平台，目前斗鱼云游戏是免费玩
  5. 电商/QQ 机器人:挂机宝
  6. 企业应用:裸金属服务器、自建机房、云服务器(ECS)
